Descrizione:

Semplice implementazione del gioco Mastermind in Flutter, dove il giocatore deve indovinare una sequenza segreta di 4 colori.

uso una lista di int per rappresentare i colori



Cronologia dei tentativi:

Classe Attempt per memorizzare i tentativi effettuati

Lista di oggetti Attempt per contenere tutti i tentativi





Variabili principali:

secret: lista di int con il codice segreto da indovinare

guess: lista di int con il tentativo corrente (inizializzato a \[0,0,0,0])

history: lista di oggetti Attempt che registra i tentativi precedenti
